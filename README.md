# 🐍 Test de Tipos de Datos en Python
## Introducción

Los tipos de datos son las distintas formas que tiene Python de almacenar información. Conocerlos es fundamental para poder crear variables, realizar operaciones y construir programas correctamente.
En este cuestionario pondrás a prueba tus conocimientos sobre algunos de los tipos de datos más importantes de Python: booleanos, enteros, cadenas de texto, listas y tuplas.

---
# 📚 Tipos de datos básicos
## ✅ Booleano (`bool`)
Representa valores lógicos y solo puede tomar dos valores: `True` o `False`.Se utiliza para expresar condiciones y tomar decisiones en los programas.
**Ejemplo:**
```python
aprobado = True
```
---
## 🔢 Entero (`int`)
Representa números enteros, es decir, números sin decimales. Se utiliza para contar elementos o realizar operaciones matemáticas.
**Ejemplo:**
```python
edad = 16
```
---

## 📝 Cadena de texto (`str`)
Representa texto y se escribe entre comillas simples o dobles. Permite almacenar palabras, frases o cualquier secuencia de caracteres.
**Ejemplo:**
```python
nombre = "Python"
```
---

## 📋 Lista (`list`)
Permite almacenar varios elementos en una misma variable utilizando corchetes `[]`. Las listas pueden modificarse después de su creación.
**Ejemplo:**
```python
colores = ["rojo", "verde", "azul"]
```
---

## 📦 Tupla (`tuple`)
Permite almacenar varios elementos en una misma variable utilizando paréntesis `()`. Las tuplas no pueden modificarse después de su creación.
**Ejemplo:**
```python
dias = ("lunes", "martes", "miércoles")
```
